# DeveloperFolio

DeveloperFolio is a modern and customizable portfolio template built with React. It helps developers showcase their skills, projects, and achievements in a professional and visually appealing way.

## Features

- Fully customizable
- Responsive design
- Integration with GitHub, Medium, and other platforms
- Easy deployment with `gh-pages`
- Built with React and Sass

## Demo

Check out the live demo: [DeveloperFolio Demo](https://your-demo-link.com)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/devinaddisondecker/developerFolio.git
   cd developerFolio
2. Install dependencies:
npm install

3. Start the development server:

npm start
4. Open your browser and navigate to http://localhost:3000.

Configuration
1. Update the profile.json file in the public directory with your personal information.
2. Customize the src folder to add your own components, styles, and assets.

Deployment
To deploy your portfolio to GitHub Pages:

1. Build the project:

npm run build

2. Deploy to GitHub Pages:

npm run deploy
